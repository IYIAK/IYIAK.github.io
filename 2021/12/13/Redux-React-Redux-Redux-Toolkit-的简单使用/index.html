个人理解，意思是，useSelector()中调用作为参数的函数 (state) => state.counter.value，然后会将 store 的整个 state 传入这个函数里，而 state 又是一个大整体，必须指明 count 是 state 中的哪一部分才行，这里就是state.counter.value（value 是在createSlice() 中定义初值时自己起的名字），然后 useSelector() 就将指明的值赋给 count。

当然啦，useSelector()是一个 hook，所以具体实现起来肯定不会这么简单，但是我觉得这么理解应该不会有啥问题。
